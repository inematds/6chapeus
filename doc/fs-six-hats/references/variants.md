# Hat order variants

Order matters. Starting with the wrong hat can sink the session.

## 1. General analysis (default)

**Order:** Blue → White → Yellow → Black → Green → Red → Blue
**Logic:** facts first, then the good, then the bad, then alternatives, then intuition. Emotion at the end so it doesn't bias the rational analysis.

## 2. Product / software feature

**Order:** Blue → White → Yellow → Green → Black → Red → Blue
**Logic:** generate alternatives (Green) before criticizing (Black), because critiques apply better to a set of options.

## 3. Interpersonal conflict

**Order:** Blue → Red → White → Yellow → Green → Black → Blue
**Logic:** emotions come FIRST because until they're out, you can't think. Then facts, common ground, new options, and only then assess risks.

## 4. Project pre-mortem

**Order:** Blue → White → Black → Red → Green → Yellow → Blue
**Logic:** Black early because the goal is to find failures. Red afterward captures internal alarms. Green looks for how to avoid the failures. Yellow at the end as a counterweight.

## 5. Creative unblocking

**Order:** Blue → Green → Green → Yellow → White → Black → Red → Blue
**Logic:** two Green passes (the 2nd with mandatory provocation) before any other mode. Black at the end so it doesn't kill options prematurely. Extra rule: Black cannot discard more than 50% of the ideas.

## 6. High-cost / irreversible decision

**Order:** Blue → White → Yellow → Black → Green → Black → Red → Blue
**Logic:** double pass through Black. The first evaluates the original proposal; the second evaluates Green's alternatives. Prevents a new alternative from escaping scrutiny.

## 7. Quick analysis (condensed)

**Order:** Blue → White → Yellow+Black (together, but visually separated) → Red → Blue
**Logic:** Green is sacrificed. Only exception to strict isolation. Use only if the user explicitly requests it. Warn: "Condensed version, without Green and without full anti-anchor."

## How to choose

1. Are there people in conflict? → **3**
2. Is it a product feature? → **2**
3. Big, irreversible commitment? → **6**
4. Want to see how it could fail? → **4**
5. Stuck? → **5**
6. In a hurry? → **7** (with warning)
7. Any other case → **1**

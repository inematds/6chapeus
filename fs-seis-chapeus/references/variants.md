# Variantes de ordem dos chapéus

A ordem importa. Começar pelo chapéu errado pode afundar a sessão.

## 1. Análise geral (padrão)

**Ordem:** Azul → Branco → Amarelo → Preto → Verde → Vermelho → Azul
**Lógica:** fatos primeiro, depois o bom, depois o ruim, depois alternativas, depois intuição. Emoção no fim para não enviesar a análise racional.

## 2. Funcionalidade de produto / software

**Ordem:** Azul → Branco → Amarelo → Verde → Preto → Vermelho → Azul
**Lógica:** gerar alternativas (Verde) antes de criticar (Preto), porque as críticas se aplicam melhor a um conjunto de opções.

## 3. Conflito entre pessoas

**Ordem:** Azul → Vermelho → Branco → Amarelo → Verde → Preto → Azul
**Lógica:** as emoções vêm PRIMEIRO porque enquanto não saem não dá para pensar. Depois fatos, terreno comum, opções novas, e só então avaliar riscos.

## 4. Pre-mortem de projeto

**Ordem:** Azul → Branco → Preto → Vermelho → Verde → Amarelo → Azul
**Lógica:** Preto cedo porque o objetivo é encontrar falhas. Vermelho depois captura alarmes internos. Verde busca como evitar as falhas. Amarelo no fim como contrapeso.

## 5. Desbloqueio criativo

**Ordem:** Azul → Verde → Verde → Amarelo → Branco → Preto → Vermelho → Azul
**Lógica:** duas passadas de Verde (a 2ª com provocação obrigatória) antes de qualquer outro modo. Preto no fim para não matar opções prematuramente. Regra extra: o Preto não pode descartar mais do que 50% das ideias.

## 6. Decisão de alto custo / irreversível

**Ordem:** Azul → Branco → Amarelo → Preto → Verde → Preto → Vermelho → Azul
**Lógica:** dupla passada pelo Preto. A primeira avalia a proposta original; a segunda avalia as alternativas do Verde. Previne que uma alternativa nova escape ao escrutínio.

## 7. Análise rápida (condensada)

**Ordem:** Azul → Branco → Amarelo+Preto (juntos, mas separados visualmente) → Vermelho → Azul
**Lógica:** sacrifica-se o Verde. Única exceção ao isolamento estrito. Usar só se o usuário pedir explicitamente. Avisar: "Versão condensada, sem Verde nem anti-âncora completa."

## Como escolher

1. Há pessoas em conflito? → **3**
2. É uma feature de produto? → **2**
3. Compromisso grande e irreversível? → **6**
4. Quer ver como poderia falhar? → **4**
5. Está travado? → **5**
6. Está com pressa? → **7** (com aviso)
7. Qualquer outro caso → **1**

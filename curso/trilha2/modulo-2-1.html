🎬 Invocando a skill

Como fazer o Claude ativar a skill certa, com contexto suficiente. Um primeiro turno bem formulado determina o resultado de toda a sessão.

💬 As frases-gatilho

A skill tem no seu description uma lista de gatilhos reconhecíveis. Usando qualquer um deles, a ativação é quase garantida.

Frases que ativam

seis chapeus six hats análise multi-perspectiva me ajuda a pensar quebra a âncora prós e contras de verdade devo fazer X? estou travado pre-mortem
2

📝 Dar contexto suficiente

Skill boa + contexto ruim = análise genérica. O que o Claude precisa saber para uma boa sessão:

✓ Contexto completo

"Quero decidir se mudo meu SaaS B2B para modelo freemium. Tenho 140 clientes pagantes (R$ 49/mês), churn de 8% ao mês, 6 meses de runway, equipe de 3. Concorrente principal fez freemium há 3 meses."

✗ Contexto pobre

"Devo fazer freemium?"

Resposta vai ser genérica. Sem preço, churn, equipe, runway — a skill não consegue ser útil.

3

🎯 Clareza da pergunta

Uma boa pergunta tem decisão mensurável no fim. "Devo X?" é boa. "O que acha?" é ruim.

4

⛓️ Explicitar restrições

Sem restrições explícitas, as alternativas do Verde viram fantasia. Diga o que você NÃO pode fazer tanto quanto o que pode.

💡 Template de restrições

  • Tempo: "Tenho 3 semanas para decidir. Não posso adiar depois de 15/abril."
  • Dinheiro: "Orçamento R$ 50k. Não posso subir."
  • Equipe: "2 devs full-time, 1 designer part-time."
  • Regulação: "Dados de saúde sob LGPD."
  • Política: "Sócio não aceita endividar."
5

🎭 Pedir variante específica

Em dúvida? Especifique a variante. "Use a variante 3 (conflito)" ou "Faça pre-mortem completo".

Quando pedir explicitamente

Quando a situação é claramente uma: "Tem conflito com minha sócia — use a variante 3."
Quando a decisão é irreversível: "Vou comprar este imóvel — variante 6 por favor."
Quando está travado: "Há 2 meses giro nas mesmas ideias — variante 5 desbloqueio criativo."
6

🚀 O primeiro turno ideal

Template que cobre tudo que você aprendeu neste módulo.

Template do primeiro turno

# Gatilho

"Me ajuda a pensar seis chapéus com quebra de âncora."

# Pergunta decisiva

"Devo [ação binária] nos próximos [prazo]?"

# Contexto

"Sou X, trabalho com Y. O problema é Z. Já tentei W."

# Restrições

"Prazo: ... | Orçamento: ... | Equipe: ... | Política: ..."

# Variante (opcional)

"Use variante X." (ou omita para o Claude escolher)

📝 Resumo do Módulo

✓
9+ frases-gatilho reconhecidas - use qualquer uma.
✓
Contexto suficiente - problema, números, histórico.
✓
Pergunta com decisão mensurável - "devo X?" não "o que acha?".
✓
Restrições explícitas - tempo, dinheiro, equipe, regulação.
✓
Variante opcional - especifique quando a situação for clara.
✓
Template reaproveitável - gatilho + pergunta + contexto + restrições + variante.
